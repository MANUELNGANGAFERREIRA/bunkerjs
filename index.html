1. O que é Lógica de Programação? A lógica de programação é a maneira como pensamos e planejamos as etapas para resolver um problema usando um computador. Imagine que você está tentando organizar um processo, como fazer uma receita de bolo. A lógica de programação é a sequência de instruções que você segue para atingir um objetivo, como o "bolo pronto"! Por exemplo, imagine que queremos organizar um conjunto de números em ordem crescente. Para isso, você segue uma sequência de passos que chamamos de algoritmo.

2. O que é um Algoritmo? Um algoritmo é simplesmente um conjunto de instruções ou passos definidos para resolver um problema ou realizar uma tarefa específica. Por exemplo, se você fosse ensinar alguém a fazer um sanduíche, você começaria com algo como: Pegue duas fatias de pão. Coloque uma fatia de queijo entre elas. Pronto, você tem um sanduíche! Esse conjunto de instruções é um algoritmo!

3. Exemplo Prático de Algoritmo: Ordenar Números Agora, imagine que temos os números: [5, 3, 8, 1, 4] e queremos ordená-los de forma crescente. Vamos criar um algoritmo simples para isso: Compare dois números consecutivos (5 e 3). O 3 é menor, então colocamos o 3 antes do 5. Compare 5 e 8. Não precisa mudar nada, pois 5 já é menor que 8. Compare 8 e 1. Coloque o 1 antes do 8. E assim por diante até que todos os números fiquem na ordem certa! Essa é a lógica de como um programa de computador pode resolver um problema de ordenação.

4. Como Programação e Algoritmos se Relacionam? Quando programamos, estamos criando algoritmos e transformando-os em código que o computador pode entender. A programação é como escrever essas instruções em uma linguagem que o computador compreende, como JavaScript, Python, ou Java.
